# Welcome to the House Analysis TWINO Test App

## Overview

The House Sales Analysis App is a web application built with Flask for analyzing and visualizing data related to house sales. The app provides various analysis results, including average price per unit, deal volume over time, and more.

### Prerequisites
- Python (3.x recommended)
- Flask
- Pandas
- Matplotlib
- Seaborn
- Chart.js

### Installation

1. Clone the repository
2. Install the required dependencies

### Usage

1. Run the Flask app: python app.py
2. Open your web browser and go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/) to access the home page.
3. Click on the provided links to view different analysis results.

### Average Price Per Unit
This page displays the average price per unit over time, considering factors such as deal, square footage, and number of bedrooms.

### Deal Volume Over Time
This page provides visualizations of how the volume of deals (number and total amount) changes over time, filtered by relevant criteria such as waterfront, view, bedrooms, and bathrooms.
